헬로 월드

display : 요소의 화면 출력(보여지는) 특성

각 요소에 이미 지정되어 있는 값
block : 상자(레이아웃) 요소 (div)
inline : 글자 요소 (span)
inline-block : 글자 +상자 요소

따로 지정해서 사용하는 값
flex : 플렉스 박스 (1차원 레이아웃)
grid : 그리드(2차원 레이아웃)
none : 보여짐 특성 없음, 화면에서 사라짐
기타 : table, table-row,table-cell 등

요소 투명도
opacity
(기본값) 1 : 불투명
0 ~1 : 0부터 1사이의 소수점 숫자
